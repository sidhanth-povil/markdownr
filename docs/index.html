Why it helps

The simple version

Most "save as Markdown" tools only work on public pages — the kind an AI could already read on its own just from a link.

Markdownr is different. Because it runs inside your browser, it can save pages that are locked to you: a paid newsletter, a journal you're logged into, an internal wiki, or an app running on localhost.

It strips the menus, ads and popups, keeps the parts that matter — headings, code, math, tables — and remembers where it came from. Then you paste it straight into ChatGPT, Claude, Obsidian, or your notes.

Feeding a page to an AI? Paste clean Markdown instead of a messy copy — the model reads it far better, and you spend fewer tokens.

Doing research? Every capture keeps the title, author, date and link, so your sources stay citable.

Care about privacy? A hosted "reader" service sees every page you send it. Markdownr never sends anything anywhere.

Questions

Good to know

Is it free?
Yes — completely free and open source under the MIT license. No paid tier, no upsell.
Do you collect any of my data?
No. Conversion happens entirely on your device and the extension makes zero network requests — no analytics, no accounts, no servers. An automated test fails the build if a network call is ever added, so the promise stays true.
Does it work on paywalled or logged-in pages?
Yes. It reads the page you're already looking at, in your own browser session — so anything you can see on screen, it can save. That's the whole point: it reaches pages a link-based tool never could.
Which browsers does it support?
Chrome and other Chromium browsers — Edge, Brave, Arc. A Firefox build is planned.
Do I need an account or sign-in?
No account, no sign-in, no email. Install it and it works.
Where does my captured text go?
Wherever you choose — your clipboard, a downloaded .md file, or a combined file when you capture several tabs. Never to a server.
How do I install it right now?
Download the release zip, unzip it, open chrome://extensions, turn on Developer mode, and click "Load unpacked." The Chrome Web Store listing is coming soon.
