What it does

Reaches pages a URL can't

Runs in your browser, in your session — so it captures logged-in pages, localhost, and intranet tools an assistant simply can't fetch.

Citation-ready frontmatter

Title, author, publication date and source URL — read from meta tags, JSON-LD, and academic citation_* tags.

Math and code, intact

KaTeX/MathJax become real $LaTeX$ instead of duplicated soup; code blocks keep their language.

Cite & tag

Copy any source as BibTeX or RIS, and add your own Obsidian-style tags to the frontmatter.

Batch a whole session

Capture every open tab into one research document — a table of contents plus each source with its URL.

Private by design

Conversion runs entirely in your browser. Zero network requests — enforced by an automated test.

Gets the details right

Math (KaTeX/MathJax)Real $E=mc^2$, not the renderer's duplicated markup.
Code blocksLanguage preserved — ```python, not a bare fence.
Hidden "related posts"Never returns a different article by mistake — captures what you're actually reading.
Cookie banners & page buildersConsent junk stripped before parsing; auto fallback when reader mode grabs too little.
