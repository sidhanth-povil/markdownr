The research loop

Capture → cite → collect

Built for building a corpus you can think with — in an LLM, in Obsidian, in a lit review.

Capture

Reader, full page, or selection — with a live preview and a token count so you know it fits the context window.

Attribute

Title, author, date and source URL land in the frontmatter automatically. Add your own Obsidian tags.

Cite

One click copies the source as BibTeX or RIS — straight into Zotero, Overleaf, or your vault.

Batch

"All tabs" folds an entire research session into one document: a table of contents plus every source.

Install

Two minutes, no account

Chrome Web Store listing coming soon. In the meantime, grab the build or run from source.

From the release build

  1. Download the latest release zip and unzip it.
  2. Open chrome://extensions and turn on Developer mode.
  3. Click Load unpacked and pick the unzipped folder.

From source

git clone https://github.com/sidhanth-povil/markdownr
cd markdownr
npm install
npm run build
# chrome://extensions → Developer mode
# → Load unpacked → build/chrome-mv3-prod
